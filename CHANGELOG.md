# Changelog

## [1.0.0] - 2026-06-15

### Added
- Visualización de fotos y videos locales usando `photo_manager`
- Carga de archivos a servidor remoto
- Visor a pantalla completa con navegación entre archivos
- Soporte para reproducción de video local y en streaming
- Gestión de contenido en nube (subir, eliminar, compartir)
- Integración con `share_plus` para compartir archivos
- Interfaz con pestañas LOCAL / NUBE
